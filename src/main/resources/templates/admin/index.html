newbee-mall The project is a set of e-commerce systems, including newbee-mall mall system and newbee-mall-admin mall background management system, based on Spring Boot 2.X And related technology stack development.  The front-end mall system includes modules such as home portal, product classification, new product launch, home carousel, product recommendation, product search, product display, shopping cart, order settlement, order process, personal order management, member center, help center and other modules.  The background management system includes modules such as data panel, carousel management, merchandise management, order management, membership management, classification management, and settings.
